## ðŸ”§ **Windows Installation**

### **Automatic Installation**
The launcher will automatically install wxPython if it's missing:
```bash
python3 launch_anora.py
```

### **Manual Installation**
If automatic installation fails:
1. **Open Command Prompt as Administrator**
2. **Run**: `pip install wxPython`
3. **If that fails, try**: `pip install --user wxPython`
4. **Or download from**: https://www.wxpython.org/download.php

### **Troubleshooting**
- **"wxPython not found"** - Run the launcher to auto-install
- **"Permission denied"** - Run Command Prompt as Administrator
- **"Build failed"** - Install Visual Studio Build Tools
- **"Import error"** - Restart Command Prompt after installation
