## üöÄ **Quick Start**

### **Installation**

1. **Download the project** or clone from GitHub
2. **Run `run.bat`** - This will automatically:
   - Install Python if needed
   - Install all dependencies
   - Launch Nova Explorer

### **Building EXE**

1. **Run `build.bat`** - This will:
   - Install build dependencies
   - Create a standalone EXE
   - Place it in the `dist` folder

### **Unity Integration**

1. **Open Unity**
2. **Go to Edit > Preferences > External Tools**
3. **Set External Script Editor** to your NovaExplorer.exe
4. **Enjoy seamless Unity development!**
