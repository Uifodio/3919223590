Quickstart:

1) Open with Unity 2022.3 LTS or newer. Install TMP Essentials when prompted.
2) Use menu Tools > Chess > Create Sample Scene. Save the scene.
3) Press Play. Click a piece, then a highlighted target square to move. Use Undo/New Game buttons.
4) Adjust AI depth in GameRoot > GameManager component (higher = stronger, slower).
5) Autosave restores your last game on next launch. Save file is at Application.persistentDataPath.
